# 安全工具补全

一个常用安全工具设计的高效Shell补全脚本，旨在提升安全从业人员的工作效率、幸福度。

支持**Bash** 和 **Zsh**。

## 功能特点

- **广泛支持**：覆盖多种常用安全工具的补全功能，详见[支持列表](/support_list.md)。
- **兼容多种Shell**：支持 **Bash** 和 **Zsh**。
- **集成支持**：支持**OhMyZsh**,**Zinit** 。
- **跨平台支持**：支持 **KaliLinux**,**macOS**
